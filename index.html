2 hour workshop

Build your first website!

1. Let's set up

  1. Start Sublime Text. It will open a new black window.
  2. Create a folder landingpage on your Desktop. Drag & drop this folder in the Sublime Text window.
  3. In Sublime Text left navigation:
    • Right click "New file"
    • Save it as index.html with Cmd + S or Ctrl + S
    • Do the same to create a style.css file
    • Then "New Folder" and create an images folder
  4. Finally double click on index.html to open it with Chrome

2. Add some HTML content

  • To save time, just right-click on the following images and save them in your images folder
  • Then start with this HTML code in index.html
<html>
  <head>
    <meta charset="utf-8">
    <title>Values at Satair</title>
  </head>
  <body>
    <h1>Our values</h1>

      <a href="#">Start now</a>
    </p>

    <img src="images/focus.jpg" alt="picture description" width="100">

3. CSS for fonts & colors

Link your stylesheet style.css in the head section of your HTML

<head>
  <link href='style.css' rel='stylesheet'>
</head>

Then add some CSS code in style.css. For some inspiration see below :-)

  • You can find nice colors on Coolors or Color Hunt
  • The official Satair color is #B11116
/* style.css */
body{
  font-size: 15px;
  font-family: Helvetica;
}
h1{
  font-family: Courier;
  color: red;
  font-size: 20px;
}
h2 {
  font-family: Courier;
  color: pink;
  font-size: 18px;
}
a {
  color: green;
  text-decoration: none;
}
a:hover {
  color: purple;
}

5. Name your tags

  • Name your header and footer with id like <div id="header"> and <div id="footer">
  • Name your values with class like <div class="value">
  • Then add this nice CSS code
#header{
  text-align: center;
  background-image: url('http://resources.mynewsdesk.com/image/upload/c_limit,dpr_2.0,f_auto,h_700,q_auto,w_670/c0nw0keg8aajgdojgn5u.jpg');
  background-size: cover;
  padding: 150px;
  color: white;
  text-shadow: 1px 1px 5px black;
}
#header h1 {
  color: white;
  font-size: 50px;
}
#header p {
  color: white;
  opacity: 0.8;
  font-size: 30px;
}
#footer{
  padding: 30px;
  background: rgb(30, 30, 30);
  color: rgb(240,240,240);
}
.value{
  padding: 50px;
  text-align: center;
}

7. Bootstrap setup

Link Bootstrap stylesheet in the head section of your HTML before your stylesheet link

<head>
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css">
</head>

Play with class="btn btn-success" and class="list-inline", your first Bootstrap classes
